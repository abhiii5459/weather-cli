weather-cli
=================
Check the weather for your city from your terminal

### Install
```shell
$ npm install -g weather-cli
```

```shell
$ weather-cli --help

  Usage
    weather-cli [input]

  Options
    city [Default: Dhaka]
    country [Default: Bangladesh]
    scale (C/F) [Default: Celcius]

  Examples
    $ weather-cli London UK C
    London, UK
    Condition: Partly Cloudy
    Temperature: 32C
```
